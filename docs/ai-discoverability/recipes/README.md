# Recipes

Recipes are short, executable guides for common AI-agent workflows. Each recipe
should name:

- the user intent,
- the primary catalog ID,
- required repos,
- required manifests,
- commands to run,
- verification gates,
- failure modes and routing for missing capabilities.
